\# Projet IA - Prédiction Âge Biologique



\## Description

Projet de machine learning visant à prédire l'âge biologique à partir de données de méthylation ADN (CpG).



\## Structure

\- feature\_selection.py : sélection des CpG

\- build\_top\_dataset.py : construction dataset

\- modeling\_notebook.ipynb : entraînement des modèles

\- age.py : script principal



\## Données

Les datasets volumineux ne sont pas inclus.
